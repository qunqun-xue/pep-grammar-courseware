# PEP小学高年级英语语法系列课件包

## 1. 项目定位
- 适用对象：小学高年级（建议五、六年级）
- 教材口径：人教PEP
- 单课时长：30分钟
- 使用场景：学生自学、教师投屏讲解、课后复习
- 难度目标：校内同步 + 单元测
- 讲解语言比例：中英约 70/30

## 2. 交付内容（对应你的3步要求）
1. 教学大纲：见 `01_教学大纲.md`
2. 20节课程稿：见 `02_课程稿_20课_学生自学版.md`
3. 图文趣味化模板：见 `03_图文与趣味化呈现模板.md`

## 3. 建议使用方式
1. 每课按统一接口学习：`WarmupScene -> RuleCard -> PitfallBox -> PracticeA/B/C -> FunMission -> ExitTicket`
2. 建议每周2课，10周完成一轮。
3. 每完成4课做1次小复盘（可使用L04/L08/L13/L20中的复盘题）。
