# 网页自学材料使用说明

## 直接打开（推荐）
1. 用浏览器打开 `index.html`。
2. 使用左侧目录、底部“上一课/下一课”翻页学习（每课一页）。
3. 支持地址栏 hash 直达，例如：`index.html#L08`。
4. 每课都有“智能批改闯关区”：可输入答案、提交批改、查看参考答案。
5. 页面里的 A/B/C 拓展题保留为开放训练；自动判题统一在“智能批改闯关区”。

## 文件说明
- `index.html`：自学主页面（单页应用）。
- `styles.css`：视觉样式与移动端适配。
- `app.js`：翻页、目录跳转、自动判题、改错批改逻辑。
- `lessons-data.js`：20课课程数据（由课程稿自动提取）。
- `quiz-bank.js`：20课自动批改题库与答案。
- `课程链接清单.md`：20个直达链接。
